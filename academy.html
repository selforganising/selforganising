Coding the web

Proposed course content

Web fundamentals
Codecademy's gentle introduction to HTML and CSS. Self-contained, interactive. Could probably be done in a few hours.
JavaScript
Simple and quick introduction to Javascript.
Introducing jQuery
The third of the three Codecademy courses that together are a good introduction to building web pages.
Introduction to Computer Science
Udacity's introductory course teaches the fundamentals of programming through building a web search engine in Python. It consists of 11 lessons, which can probably each be completed in a day or so.
Web Development
Another Udacity course. This one zips through the principles of how the web works and then moves onto building a real-world web application in Python using Google App Engine. There are 7 lessons, which could require a day each.
A real-world project
Constructing a website for a Collective member or for a Camden BID member or for another business or voluntary group.
